NAME
asmlib_tutor - examples of AsmLib usage and discussion

PURPOSE
This package contains a collection of source files and
discussion of each file.  It is intended to introduce the
following topics:

 * terminal displays
 * memory orginization and usage
 * passing parameters to programs
 * starting and controlling a process
 * sending and recieving signals
